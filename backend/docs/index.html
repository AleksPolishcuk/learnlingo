LearnLingo API (1.0.0)

Download OpenAPI specification:

LearnLingo Team: support@learnlingo.dev License: MIT

REST API for the LearnLingo language-tutoring platform.

Authentication

Most endpoints require a Bearer JWT token. Obtain one via POST /api/auth/register or POST /api/auth/login, then supply it on every protected request:

Authorization: Bearer <token>

Error shape

All error responses share the same envelope:

{ "message": "Human-readable description" }
